### About
Kittygram - проект позволяющий пользователям делиться фотографиями и достижениями своих пушистых компаньонов. 
Проект предусматривает регистрацию пользователей, поддержание базы данных, администрирование ресурсов и
подготовлен для разворачивания как на локальном, так и на удаленном сервере.


### TechStack
Использованные технологии:
Python
Django
Nginx
Docker
PostgreSQL
